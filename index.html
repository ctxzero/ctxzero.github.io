Case Studies

Real findings, real fixes

Auth Bypass — SQLi (Web App)

Auth endpoint injection allowed login bypass via crafted payloads. Delivered PoC, fixed parameterized queries and rate limiting.

Impact: Account takeover & data access. Fix: Parameterized queries, account lockouts, monitoring.

IDOR — Broken Authorization

Insecure direct object reference exposed user PII. Added authorization checks and standardized object IDs.

Impact: Sensitive data disclosure. Fix: Proper access controls, logging & alerts.

Misconfigured service accounts enabled Kerberoasting & lateral movement. Delivered detection playbook and ACL hardening steps.

Impact: Domain compromise risk. Fix: Harden service accounts, rotate creds, monitor ticket requests.
